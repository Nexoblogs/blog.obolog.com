Obolog ha cerrado

Durante el año 2002 y 2020, Obolog ofreció de manera gratuita blogs para la comunidad hispanoablante. En junio del 2020, el proyecto se descontinuó.

Problemas de acceso a la web durante las últimas horas

Thu, Feb 23, 2006

Es probable que algunos de vosotros hayáis experimentado algunos problemas de acceso a la web: lentitud, errores e incluso falta de conectividad. En estos momentos estamos revisando el sistema para verificar su estabilidad.

Agradecemos vuestra comprensión y paciencia y sentimos las molestias que esto os haya podido causar.
