Obolog ha cerrado

Durante el año 2002 y 2020, Obolog ofreció de manera gratuita blogs para la comunidad hispanoablante. En junio del 2020, el proyecto se descontinuó.

Blog Action Day - ¡Hazte oir!

Mon, Oct 15, 2007

Como anunciamos hace unos días, hoy se llevará a cabo el Blog Action Day. Durante todo el día miles de blogs unirán sus voces entorno a un mismo tema: el medio ambiente. Tú puedes colaborar desde tu blog, escribiendo un artículo que de algún modo esté relacionado con la temática y que fomente la concienciación o el debate medioambiental.

Para participar tan sólo tienes que usar la etiqueta "Blog Action Day" a tu mensaje. Todos los mensajes aparecerán en esta lista. No dejes pasar la oportunidad. Si tienes un blog, ¡hazte oir!
