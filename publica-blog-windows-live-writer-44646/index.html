Publica en tu blog desde Windows Live Writer

Thu, Dec 6, 2007

Como algunos sabréis, OboLog os ofrece la posibilidad de enviar contenidos a vuestros blogs desde aplicaciones externas ( Flickr, DailyMotion, etc... ) a través del MetaWeblog API. En esta ocasión vamos a explicar cómo sacar partido de esta potente API para poder escribir posts en vuestro blog desde una aplicación de escritorio: Windows Live Writer.

  1. En primer lugar deberéis descargar Windows Live Writer. Podéis hacerlo desde aquí.
  2. Una vez lo hayáis descargado y hayáis ejecutado el instalador, deberéis escoger "Another Weblog Service" ( "Otro servicio de blogs" )
  3. Introducid la dirección de vuestro blog ( http://vuestroblog.obolog.com ) y escribid el usuario y la contraseña con la qué accedéis a vuestro blog.
  4. En la siguiente pantalla deberéis indicar que OboLog usa el MetaWeblog API, y le indicaréis que ésta se encuentra en la dirección http://www.obolog.com/xmlrpc
  5. En caso de tener más de un blog, os preguntará cuál de ellos queréis vincular a Windows Live Writer.
  6. Voilà: Windows Live Writer enviará un post de prueba a vuestro blog y podréis empezar a escribir sin necesidad de conectaros a la web.

Escribir desde esta aplicación de escritorio tiene algunas ventajas:

  • Podéis escribir anotaciones sin necesidad de estar conectados a internet. Las guardáis como borradores y las publicáis en cuanto tengáis conexión.
  • Os permite previsualizar el mensaje tal y como quedaría en vuestro blog, sin necesidad de publicarlo.
  • Incorpora un editor de imágenes.

Aunque sin embargo también deberéis tener en cuenta que:

  • Sólo podréis publicar imágenes previamente cargadas en un servidor externo.
  • Deberéis seleccionar siempre una fecha de publicación. En caso contrario, lanzará un error.
  • No podéis seleccionar categorías o tags para el mensaje.
